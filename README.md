# Pizzaria 👨‍🍳

Repositório da api do projeto da pizzaria 🍕

## 🚀 Tecnologias

- [Node](https://nodejs.org/en/)
- [Express](https://expressjs.com/pt-br/)
- [Mongoose](https://mongoosejs.com/)
- [Socket.IO](https://socket.io/)

## 🔥 Instalação

```bash
# backend
$ yarn install
$ yarn dev
```
